App Description: This website is a small pseudo-magazine that displays an assortment of cocktails whenever the page is refreshed.
    Additionally, there is a social aspect to this page, where someone can post anonymously(or not) what their preferences
    are for both liquor and for a cocktail, thus displaying it for others to notice and look up themselves.

API Link: The Cocktail DB: https://rapidapi.com/thecocktaildb/api/the-cocktail-db/
